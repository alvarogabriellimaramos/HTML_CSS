Formas de usa o css

Estilo inline

o estilo inline é uma forma de atribuição ao invés da tag utilizamos o atributo style,exemplo <body style= "backgroundColor ="blue"> </body>

Estilo internal/locais

o estilo internal é uma forma de estilo mas recomedada do que o inline,
já que se torna mais organizada,mas ainda não é mais eficiente e comparada a external não é muito recomedada ,
utilizamos a tag <style> </style> para criamos nossas css,lembrando que essa tag tem que está dentro da tag head

Estilo external

é a forma mais eficiente que eu conheço,criamos uma tag chamada link e fora do codigo html criamos um arquivo com a extensão css e colocamos o nome do arquivo css dentro da tag link=href=''
