Lista em HTML

Lista não ordenada

As lista não Não ordenadas são lista onde a ordem dos elementos dentro dela são muito importate,elas são chamadas pela tag <ol> </ol>

Exemplo

  1. Item 1
  2. Item 2
  3. Item 3

Lista não ordenadas

Lista não ordenadas são lista em que a ordem não tem muita importancia,são criandas a parti da tag <ul> </ul>

Exemplo

  • Item 1
  • Item 2
  • Item 3

Lista mistas

Tambem podemos criar lista dentro de outras listas por exemplo

Exemplo

  1. Python
    • A linguagem python server para criar jogos
    • A linguagem python server para criar IAS
  2. JavaScript
    • A linguagem JavaScript server para criar site na web
    • A linguagem JavaScript server para criar aplicativos webs
  3. Csharp
    • A linguagem Csharp server para criar jogos
    • A linguagem Csharp server para criar apis
